Para criar um projeto com vite, comando:
    npm create vite@latest

Logo em seguida, pedirá para fazer a instalação dos pacotes do vite. Digite o que se pede, y ou n.
Depois digite o nome do projeto.
Escolha a opção React
Escolha a opção Javascript
Caminhe até a pasta do projeto com cd
Siga as orientações. Primeiro cd nome-do-projeto, npm install e em seguida npm run dev.

Depois o projeto irá rodar, como se tivesse no Live Server.

Quando for enviar o site para o Github, apagar a pasta node_modules.

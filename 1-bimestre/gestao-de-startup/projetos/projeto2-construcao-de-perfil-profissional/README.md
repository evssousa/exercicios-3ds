# 📄 PROJETO 2 – CONSTRUÇÃO DE PERFIL PROFISSIONAL (CURRÍCULO + LINKEDIN)

## 🧩 PROBLEMA

Você está prestes a buscar sua **primeira oportunidade na área de tecnologia** (estágio, jovem aprendiz ou emprego).

Uma empresa chamada **TechStart** está analisando candidatos iniciantes para futuras oportunidades.

Porém, há um problema:

> Muitos candidatos não sabem se apresentar profissionalmente, não possuem currículo adequado e não têm presença no ambiente digital profissional.
> 

Sua missão será resolver esse problema.

## 🎯 PROJETO

Você deverá:

> Criar um **currículo profissional** e um **perfil no LinkedIn** (ou planejamento dele), como se estivesse se candidatando a uma vaga na área de tecnologia.
> 

## 🧠 REQUISITOS

### 📄 1. Currículo

Seu currículo deve conter:

- Nome completo
- Contato (email ou telefone fictício)
- Objetivo profissional
- Formação
- Habilidades
- Projetos ou experiências (mesmo que escolares)

📌 Regras:

- Deve ser **claro e organizado**
- Deve ser **objetivo (sem textos muito longos)**
- Linguagem **profissional**

OBS: o currículo deve ser feito em sala.

### 💼 2. Perfil profissional (LinkedIn)

Você deverá:

👉 Criar ou planejar um perfil contendo:

- Nome
- Título profissional
- Resumo (quem você é profissionalmente)
- Habilidades
- Projetos

📌 Pode ser:

- Criado diretamente no computador
    
    OU
    
- Escrito no caderno/folha como rascunho completo

OBS: Além de ser feito em outro momento (não é para ser feito em sala), caso já tenha uma conta no LinkedIn, você irá refatora-lo!
